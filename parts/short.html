Тест по темпористике

Тест по новой типологии личности — темпористике, разработанный одним из ее авторов — Александром Латышевым. Тест содержит 54 вопроса, на каждый из которых возможно три варианта ответа «Да», «Не знаю/Не важно» и «Нет». По результатам теста будет показано количество баллов для каждого темпористического типа. Тип, набравший максимальное количество баллов, следует считать основной версией.

Подробнее о темпористике можно узнать на сайте ТЕМПОРИСТИКА.РФ.

Для выбора одного из вариантов ответов просто щелкните по нему. Если вы ошиблись с выбором, нажмите кнопку «Назад» в броузере. Если вы хотите отложить прохождение теста, сохраните полный адрес страницы (включая символы после #). Открыв эту ссылку, вы продолжите прохождение с того же самого вопроса.

Как искать совместимых людей?

  1. Вперевую очередь должен инетересовать тип по Темпористике. Если интересует долгосрочное сотрудничество, то это совмеситмость ключевая. Так как идея новая, то изначално можно искать людей в группах по темпористике. Я же хочу создать сообщество знакомств по психологическому расчету, для поиска совместимых людей.
  2. Следующим типом должен быть психософский тип. Если есть совпадение по психософии вы можете в быту без особых проблем что-то делать. Иначе делиться на подгруппы, искать бытовые и тактические комромиссы.
  3. Соционика дает понимания фон взаимодейсвия.

Что это дает человеку?

  1. Решение проблемы доверия, быстрый контроль адеватности человека (Темпориситка это стратегия жизни человека, Психософия как будет реализовывать свою стратегию, Соционика - за счет какой информации и какие акценты.)
  2. Если собрать сообщестов (см. внизу страницы) открывает новые возможности для поиска кадров. Поиска знакомсв и связей.
  3. Общение с дополняющим человеком помогает привести состояние человека в норму. Естесвенная психотерапия.
  4. Роботосопобность и слаженость работы дополняющих людей сверхэффективна.
  5. От дополняющего человека: Мотивация, воодушевление, муза, правильные слова, наставления и советы.
  6. Cаморазвите, самопонимание, какое влияние оказывает человек на окружающих, окружающие оказывают на него.
  7. Борьба с информационной пропагандой и и нформационным мусором. Определение дейсвительно важных для себя вещей.
